# 操作系统类：软件包

## windows

* chocolatey

* PowerShell的[PackageManagement模块](https://technet.microsoft.com/en-us/library/dn890706.aspx)

获取所有可用包源：

``` powershell
Get-PackageSource
```

获取所有可安装的包：

``` powershell
Find-Package -ProviderName chocolatey
```

包管理相关命令查询：

``` powershell
man package
```
